🚀 QUICK START

Terminal 1 - Ollama:
    $ ollama serve
    (First time: ollama pull phi)

Terminal 2 - Backend:
    $ cd backend
    $ python3 -m venv venv
    $ source venv/bin/activate
    $ pip install -r ../requirements.txt
    $ python main.py

Terminal 3 - Frontend:
    $ cd frontend
    $ npm install
    $ npm run dev

Then visit: http://localhost:3000
